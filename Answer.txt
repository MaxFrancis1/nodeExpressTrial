## Copilot
Creating shared HTML partials for the header, navigation, and footer can help you maintain consistency across your web pages and make updates easier. Here's how you can do it using Express and EJS:
Step 1: Create Partial FilesFirst, create separate EJS files for your header, navigation, and footer. For example:
views/partials/header.ejs

<header>
    <h1>My Website</h1>
</header>

views/partials/nav.ejs

<nav>
    <ul>
        <li><a href="/">Home</a></li>
        <li><a href="/about">About</a></li>
        <li><a href="/contact">Contact</a></li>
    </ul>
</nav>

views/partials/footer.ejs

<footer>
    <p>&copy; 2024 My Website</p>
</footer>

Step 2: Include Partials in Your Main LayoutNext, create a main layout file where you will include these partials. This layout will be used by your other views.
views/layout.ejs

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title><%= title %></title>
</head>
<body>
    <% include partials/header %>
    <% include partials/nav %>
    
    <main>
        <%- body %>
    </main>
    
    <% include partials/footer %>
</body>
</html>

Step 3: Use the Layout in Your ViewsFinally, use this layout in your individual views. For example, your about.ejs might look like this:
views/about.ejs

<% layout('layout') %>
<h2>About Us</h2>
<p>Welcome to the about page!</p>

Step 4: Set Up Your RoutesMake sure your routes render the views correctly.
routes/about.js

var express = require("express");
var router = express.Router();

router.get("/", (req, res) => {
    res.render("about", { title: "About Us" });
});

module.exports = router;

app.js

var express = require("express");
var aboutRouter = require('./routes/about.js');

var app = express();
app.set("views", "views");
app.set("view engine", "ejs");
app.use(express.json());
app.use(express.urlencoded({ extended: true }));
app.use(express.static("public"));

app.use("/about", aboutRouter);

app.listen(3000, () => {
    console.log('Express is running on port 3000');
});

This setup allows you to maintain a consistent header, navigation, and footer across all your pages, making your code more modular and easier to manage.
